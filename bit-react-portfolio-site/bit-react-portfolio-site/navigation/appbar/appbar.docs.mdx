A component renders an app bar (navigation bar) for the portfolio

### Component usage

Use with `ThemeProvider` for a theme object.

```js
<Appbar />
```

### Render hello world!

```js live
<ThemeProvider theme={darkTheme()}>
  <Appbar />
</ThemeProvider>
```
